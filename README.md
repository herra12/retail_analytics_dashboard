# Retail Sales Analysis & Power BI Dashboard with Automated Pipeline

## Objective
This project aims to analyze retail sales, structure data 
in a relational database, create a Power BI dashboard, 
and automate customer subscription status prediction.

## Project Steps
1. Retail data cleaning and preparation
2. Dimensional modeling (dimension tables and fact table)
3. Data storage in MySQL
4. Power BI dashboard creation
5. Subscription_status prediction with Machine Learning
6. Pipeline automation with Apache Airflow

## Technologies Used
- Python
- SQL (MySQL)
- Power BI
- Scikit-learn
- Apache Airflow

## Pipeline Automation
Apache Airflow is used to orchestrate the entire data pipeline.
The DAG automates the following steps:
- Feature engineering
- Machine Learning model training
- Customer subscription status prediction generation
This automation ensures a reliable, reproducible workflow
ready for scheduled execution.
